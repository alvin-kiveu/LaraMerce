## Installation

You can install the package via composer:

```bash
composer require LaraMerce/LaraMerce
```

Publish the config file with:

```bash
php artisan vendor:publish --provider="LaraMerce\LaraMerceServiceProvider" --tag="config"
```

Usage

```php
Route::LaraMerce();
```
Testing
```bash
composer test
```
